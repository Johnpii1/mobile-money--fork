Quick Installation:
  $ cd cli && npm install && npm run build

Basic Commands:
  $ momo-cli dashboard              # Display dashboard once
  $ momo-cli dashboard --watch      # Auto-refresh (5s interval)
  $ momo-cli dashboard:live         # Compact live monitor
  $ momo-cli dashboard:export       # JSON export for scripting

Advanced Usage:
  $ momo-cli dashboard -w -i 3000   # Refresh every 3 seconds
  $ momo-cli dashboard:live -i 2000 # Live monitor every 2 seconds

Integration:
  $ METRICS=$(momo-cli dashboard:export)
  $ FAILED=$(echo $METRICS | jq '.queue.failedJobs')
  $ echo "Failed jobs: $FAILED"
